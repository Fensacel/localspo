# Coding Rules

Semua kode menggunakan TypeScript Strict.

Gunakan Functional Component.

Tidak boleh menggunakan Class Component.

Gunakan React Hooks.

Gunakan Zustand.

Gunakan Tailwind.

Semua Service dipisahkan.

Semua fitur harus modular.

Jangan membuat file lebih dari 300 baris jika bisa dipecah.

Gunakan nama yang jelas.

Contoh

MusicScannerService

LyricsService

PlayerStore

ThemeProvider

---

Semua fungsi harus memiliki satu tanggung jawab.

Gunakan SOLID.

Gunakan Clean Architecture.

Gunakan reusable component.

Selalu lakukan lint sebelum commit.

Gunakan ESLint dan Prettier.
