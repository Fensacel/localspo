# bluetune

bluetune adalah aplikasi desktop premium untuk memutar koleksi musik lokal.

## Features

- FLAC
- MP3
- Lyrics
- LRC
- Playlist
- Equalizer
- Dynamic Theme
- Waveform
- Visualizer
- Audiophile Info

## Tech

Electron

React

TypeScript

TailwindCSS

Framer Motion

WaveSurfer

Web Audio API
